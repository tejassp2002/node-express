This model is created using Teachable Machine Pose model which I have trained on around 800 images per class of 4 different Hand Gestures. I have used Teachable Machine Pose Library for creating this web application ( i.e. loading the model, accesing the webcam and finally predicting the hand gesture). This model is built on PoseNet model layer and Teachable Machine Model layer which I have trained via Teachable Machine Model
